DOES THE SEC TRULY HAVE THE BIGGEST HOME FIELD ADVANTAGE IN COLLEGE FOOTBALL?

College football isn’t just a game, it’s a thrilling experience that draws the attention of so many. With these exciting games and atmospheres, the home field advantage becomes more and more of a factor.

Many teams enjoy a unique advantage when playing on their home field in college football. Let’s look at some numbers to explore why the SEC is named to have the biggest home field advantage compared to other conferences.

In fact, does the SEC have the biggest home court advantage?

When looking at some data sets from the college football season of 2022, it becomes clear that the SEC conference stands out in some aspects.

The first question is going to be how do college football teams perform in terms of points scored in home and away games, and does this show anything significant? Here is a breakdown off the average points scored by each conference at home and away.

Code 
library(tidyverse)
library(ggbump)
library(gt)
library(wehoop)
library(patchwork)

football <- read_csv("footballlogs22.csv")

Pointsscored <- football |>
  group_by(Conference) |>
  summarize(PointsscoredHomeAway = sum(TeamScore + OpponentScore))

home <- football |>
  filter(is.na(HomeAway) == "TRUE")

OppscoreH <- football |>
  group_by(Conference) |>
  filter(is.na(HomeAway) == "TRUE") |>
  summarize(OpponentScoreHome = mean(OpponentScore))

homepoints <- home |>
  group_by(Conference) |>
  summarize(Pointsathome = mean(TeamScore))


away <- football |>
  filter(HomeAway == "@")


OppscoreA <- football |>
  group_by(Conference) |>
filter(HomeAway == "@")
  

awaypoints <- away |>
  group_by(Conference) |>
  summarize(Pointsaway = mean(TeamScore))

ggplot() + 
  geom_bump(data=Pointsscored, aes(x=PointsscoredHomeAway, y=Conference, color=Conference)) +
  geom_bump(data=homepoints, aes(x=Pointsathome, y=Conference, color=Conference)) +
  geom_point(data=homepoints, aes(x=Pointsathome, y=Conference, color=Conference), size = 4) +
  geom_bump(data=awaypoints, aes(x=Pointsaway, y=Conference, color=Conference)) +
  geom_point(data=awaypoints, aes(x=Pointsaway, y=Conference, color=Conference), size = 2) +
  
  labs(
  title="Average points scored in college football", 
  subtitle="Big cirlce are points scored at home. Small circle are points away",
  caption = "Source: Sports Reference | By Nic Wiedenhorn",
   y= "Conference", 
  x = "Points scored") +
  theme_minimal() +
  
  
  theme(
    plot.title = element_text(size = 16, face = "bold"),
    axis.title = element_text(size = 8), 
    plot.subtitle = element_text(size=10), 
    panel.grid.minor = element_blank(),
    plot.title.position = "plot"
    )

This chart reveals the average points scored at home and the points scored away for each conference. It shows that the SEC is one of the highest scoring conferences at home, which shows dominance on their home turf.

Some other teams might have a bigger point margin or points scored at home but to look for a bigger indicator, let’s have a look at the number of games won at home and away in the SEC.

Is there a significant difference between the performance of the SEC compared to any other conference when playing at home or away?

Totalgames <- football |>

  group_by(Conference) |>
  summarize(TotalGames = n())
  

Home <- football |>
  filter(is.na(HomeAway) == "TRUE")

  group_by(Conference) |>
  filter(Outcome == "W") |>
   summarize(HomeWins = n()) 

bar2 <- ggplot() + geom_bar(data=Homewins, aes(x=reorder(Conference,HomeWins),weight=HomeWins)) +
  coord_flip()

  group_by(Conference) |>
  filter(HomeAway == "@", Outcome=="W") |>

bar3 <- ggplot() + geom_bar(data=Away, aes(x=reorder(Conference,AwayWins),weight=AwayWins)) +

SEC <- Homewins |> filter(Conference == "Southeastern Conference")

These bar charts represent the amount of games won at home and games won on the road. The red bar represents the SEC conference and shows how much they stand out at home. This shows the hard challenge for teams in the SEC playing away.

What conclusions can be drawn when looking at the win percentage of the SEC and other conferences at home and away games, and does this support the challenge of playing at the SEC?

Code 
home_data <- football |>
  filter(is.na(HomeAway) == "TRUE") |>
  group_by(Conference) |>
  summarize(HomeWins = mean(Outcome == "W"))

away_data <- football |>
  filter(HomeAway == "@") |>
  group_by(Conference) |>
  summarize(AwayWins = mean(Outcome == "W"))

confwins <- home_data |> inner_join(away_data)

confwins |>
  gt() |>
  cols_label(
    HomeWins = "HomeWinPercentage",
    
    AwayWins = "AwayWinPercentage",
    
  ) |>
    fmt_percent(
    columns = c(HomeWins,AwayWins),
    decimals = 1
  ) |>
  tab_header(
    title = "Is the SEC the hardest conference to play away?",
    subtitle = "The SEC has the highest home win percenate out of every conference."
  ) |> tab_style(
    style = cell_text(color = "black", weight = "bold", align = "left"),
    locations = cells_title("title")
  ) |> tab_style(
    style = cell_text(color = "black", align = "left"),
    locations = cells_title("subtitle")
  ) |>
  tab_source_note(
    source_note = md("Source: Sports Reference | By Nic Wiedenhorn")
